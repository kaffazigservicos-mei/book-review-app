# 📚 App de Comentários de Livros

Este projeto é um aplicativo interativo criado com **Python** e **Streamlit** para analisar o Top 100 de livros mais populares no marketplace da Amazon.
Foi criado para testar conhecimentos iniciais do Python

## 🚀 Como acessar o projeto
Você pode visualizar o app online através deste link: 
[https://streamlit.app](https://streamlit.app)

## 🛠️ Tecnologias Utilizadas
- **Python**: Linguagem principal.
- **Streamlit**: Para a criação da interface web.
- **Pandas**: Para manipulação e limpeza dos dados.
- **Plotly**: Para a criação de gráficos interativos.

## 📊 Funcionalidades
- Visualização dos 100 livros mais populares.
- Gráficos interativos de avaliação e preços.
- Filtros por categoria ou autor (se houver).

## 📁 Estrutura de Arquivos
- `app.py`: Página inicial do aplicativo.
- `pages/`: Outras páginas do app.
- `datasets/`: Base de dados em CSV utilizada.
